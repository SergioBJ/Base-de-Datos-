# Base de Datos - Tienda (SQL Server)

Proyecto práctico de base de datos usando SQL Server.

## Descripción
Este proyecto simula una base de datos para una tienda, donde se gestionan:
- Clientes
- Productos
- Pedidos
- Detalle de pedidos

Incluye scripts para:
- Creación de la base de datos y tablas
- Inserción de datos
- Consultas SQL con JOIN, GROUP BY y funciones de agregación

## Tecnologías
- SQL Server
- T-SQL
